## 🚀 FEATURED PROJECTS

### 🟣 **UniFocus**  
*Full-stack productivity & habit tracking suite*

Comprehensive personal dashboard combining habit tracking, to-dos, calendar, timetable, stress logging, and an animated Pomodoro timer with custom SVG liquid-fill effects. Built with a robust 28-endpoint REST API, bcrypt-secured authentication, and real-time data synchronization.

**Stack:** React · Express.js · MySQL · Vercel  
**[→ Live Demo](https://unifocus.vercel.app)** · [→ Repository](https://github.com/RaktheshTG/unifocus)

---

### 🔴 **PaperTrail**  
*RAG-based academic paper analysis platform*

Transform academic paper URLs into structured markdown summaries, interactive visual concept maps, and contextual Q&A. Eliminates flat chat interfaces with semantic understanding of research papers.

**Features:**
- PDF/Paper URL ingestion
- Structured markdown extraction
- Visual concept mapping
- Context-aware Q&A via RAG

**Stack:** React · TanStack Start · Cohere Embeddings · Pinecone · Groq LLM  
**Status:** 🚧 MVP (Pre-deployment)  
**[→ Repository](https://github.com/RaktheshTG/papertrail)**

---

### 🟣 **CustomerSupportAI**  
*Intelligent multi-agent support automation engine*

LangGraph-powered stateful workflow engine that routes customer queries across specialized agents:
- **Sales Agent** — Product recommendations & upsells
- **Technical Agent** — Troubleshooting & debugging
- **Billing Agent** — Invoicing & subscription queries
- **Account Agent** — Profile & preference management

RAG-grounded evaluation nodes with strict human-in-the-loop approval for sensitive operations.

**Stack:** LangGraph · RAG · SQLite · Python  
**[→ Repository](https://github.com/RaktheshTG/customer-support-ai)**
